## 使用方法
### 1.抓包冷启动手机京东APP的cookies
### 2.初次运行软件后会在同目录下生成config.ini文件
### 3.多个cookies请在config.ini文件中按序填写
```
[account]
cookies1=pin=xxx
cookies2=pin=xxx
```
## 注意事项
### 1.请在预约开始后运行软件，否则会预约失败
### 2.软件会自动校准时差，日志中为本地时间
### 3.欢迎交流分享
